盒模型和box-sizing:border-box

默认盒模形宽高是指content的宽高
box-sizing:border-box是指content加内边距padding加边框border

css选择器种类

元素选择器
div{}
类选择器
.setion{}
ID选择器
#cat{}
通配选择器
*{} // 所有html元素
属性选择器
[class^=icon-] 属性有class 并且 class的值以icon-开头
a[href*=baidu.com] 属性中有href的a标签，并且值包含baidu.com

关联选择器

相邻兄弟选择器 只会匹配紧跟其前方元素的同胞元素（不包含自己，有一个共同的父元素）

li + li {
    color: red;
}

<ul>
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
</ul>
// 第二和第三变成红色

通用兄弟选择器

元素 ~ 元素 {样式声明 }

会匹配第二个元素,条件是它必须跟(不一定是紧跟)在第一个元素之后,且他们都有一个共同的父元素

子选择器
元素1 > 元素2 {样式声明 }

后代选择器

中间空格表示
div span { background-color: DodgerBlue; }
<div>
  <span>Span 1.
    <span>Span 2.</span>
  </span>
</div>
// div下的所有span都被选中

伪元素选择器 （伪类 :hover 伪元素::after）

// html
<ul>
    <li>li:first-child</li>
    <li>2</li>
    <li class="three">li:not(.three)</li>
    <li>li:last-child</li>
</ul>

// css
li:not(.three){
  color: blue;
}
li:first-child{
  color:yellow
}
li:last-child{
  color: red
}

:last-of-type
父元素的所有子元素中的最后一个选定元素，也包括子元素的最后一个子元素,（容易用错！）

Parent tagName:last-of-type

<div>
    <p>1</p>
    <div>
      <p>1-1</p>
      <p>1-2</p> // 这个是子元素的最后一个p标签，同样被选中
    </div>
    <p>3</p>
</div>

div p:last-of-type{
  color: blue
}

:nth-child

tr:nth-child(2n+1)
表示HTML表格中的奇数行。

tr:nth-child(odd)
表示HTML表格中的奇数行。

tr:nth-child(2n)
表示HTML表格中的偶数行。

tr:nth-child(even)
表示HTML表格中的偶数行。

span:nth-child(0n+1)
表示子元素中第一个且为span的元素，与 :first-child 选择器作用相同。

span:nth-child(1)
表示父元素中子元素为第一的并且名字为span的标签被选中

nth-child和nth-of-type区别

<section>
    <div>我是一个普通的div标签</div>
    <p>我是第1个p标签</p>
    <p>我是第2个p标签</p>  <!-- 希望这个变红 -->
</section>

p:nth-child(2){
    color: red;
}
p:nth-of-type(2){
    color: blue
}

// 第一个p标签为红色 nth-child表示子元素第二个元素，且是p标签
// 第二个p标签为蓝色 nth-of-type 表示子元素的第二个p标签。

选择器的优先级排名

元素选择器<类选择器<id选择器<内联样式
!important会覆盖其它所有，所以是最高级，但要尽量避免使用

如果优先级一样，后声明的会覆盖先声明的

css3新特性

  • 字体 font-face

  • 图片边框

    border-image:url(border.png)3030 round;

  • 渐变

    background: linear-gradient(direction, color-stop1, color-stop2,...);

  • 阴影

    text-shadow:[inset] h-shadow v-shadow blur color;

    默认外阴影，inset内阴影

    h-shadow：水平偏移 必传

    v-shadow：垂直偏移 必传

    blur：阴影宽度

    color: 阴影颜色

<div>we</div>
div{
    text-shadow:5px 6px 5px black; // 5是水平
}

text-shadow:box-shadow: 0px 0px 12px red, inset 0px 0px 12px blue; // 用逗号隔开可以叠加

反射可以达成倒影效果

  • background-image 可以多个背景图片，用,隔开

  • 过渡 transition

    transition是简写, 当div的宽度变化时就会有动画过渡效果，常见可以是:hover中改变div宽度。

    div{
    width: 100px;
    heigth: 100px;
    transition: width 2s;
}

  • 动画

@key-frames myAnimation{ // 定义一个过程
    from {background:red;}
      to {background:yellow;}
}
// from to也可以用百分比代替

@keyframes myfirst
{
    0%   {background: red;}
    25%  {background: yellow;}
    50%  {background: blue;}
    100% {background: green;}
}

div {
    width:100px;
    height:100px;
    background:red;
    animation:myAnimation 5s;
}
  • filter 滤镜
 // 模糊
img{
    filter: blur(5px)
}

CSS实现隐藏页面的方式

1、.hide { // 透明度变0 对网页布局起作用
opacity: 0;
}
2、.hide { // 对网页布局起作用
visibility: hidden; // 默认 visible
}
3、.hide{ // 影响布局，元素是真的不见了
display: none;// 默认是inline
}

实现水平居中和垂直居中

水平居中

1、text-align:center // 行内内容（例如文字）如何相对它的块父元素对齐,不能控制块级元素本身对齐，只控制它的行内内容

justify
文字向两侧对齐，对最后一行无效。

justify-all
和justify一致，但是强制使最后一行两端对齐。

2、给固定宽度，然后设置margin: 0 auto; // 上下外边距0，左右auto
3、display: flex;justify-content:center;
4、绝对定位，left:50% marign-left:-自身宽度50%
5、绝对定位，left:50% transform: translateX(-50%) // 无需知道元素宽度

垂直居中

1 display: flex;align-items:center;

2 height和line-height设置同样的高度

3 display: table-cell;vertical-align:middle;(flex下vertical-align会失效)

4 绝对定位加top:50% 再加margin偏移本身高度50%

5 绝对定位加top:50% transform: translateY(-50%) // 无需要知道元素高度

圣杯布局

三列都float:left;右左负外边距本身宽，主列内边距撑开。这样就中间宽度自适应，两边固定宽。

  padding: 0 50rpx;
  font-size: 28rpx;

  transition: all 0.5s cubic-bezier(0, 1, 0.5, 1) 0s;
