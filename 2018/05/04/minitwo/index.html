小程序异步问题

发表于 2018-05-04 | 分类于 技术

问题：全局需要请求一个配置数据（鉴权之类的），多个页面需要用到，
全局app.js在onLaunch中请求这个数据，page.js（如首页）一进页面开始请求（如同样在onLaunch周期内请求），
而且请求需要用到全局配置数据，但这时，全局数据未必返回。

聪明的，可能自己已经想到解决，社区中问题~官方解释叫你去看下javascript异步编程~（有点伤人，emmmm~~）

没想通的朋友请往下看。。。

解决方法：
app.js 请求完数据，看下全局存不存在一个回调方法，有的话，调用它，把请求的结果传进去。

page.js 查看全局的数据存不存在，存在就直接用就好，不存在呢？给全局设置一个回调方法。因为上一步
你已经设定全局如果有回调就把数据传给你。

原理如以上，代码如下：

app.js中

App({
  onLaunch: function () {
    globalFn().then(resp => {
          wx.setStorageSync('resp', resp)
          if (this.cb){
            this.cb(xData)
          }
        })
  },
)}


page.js中

Page({
   onLoad () {
       let resp = wx.getStorageSync('resp')
       if (resp) {
         // 全局的数据能获取已经能获取到，往下执行就好
       } else {
        // 没有获取到，在回调中能取到
         get().cb = cbData => {
            // cbData就是全局的数据，在这里一样往下执行就好
         }
       }
  },
})
